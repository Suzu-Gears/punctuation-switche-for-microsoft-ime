句読点スイッチャー(Punctuation Switcher)

句読点スイッチャーは、Microsoft IMEの句読点の設定を一発で切り替えるツールです。
タスクトレイアイコンのクリックかショートカットキーにより、「、。」(句点読点)「，．」(カンマピリオド)を切り替えることができます。
ショートカットキーは、変更および使用しないようにすることが可能です。

本ツールはAutoHotkey製のスクリプトです。スクリプトを実行ファイル化しており、AutoHotkeyをインストールすることなく実行可能になっています。

動作環境
OS: Windows 10/11

インストール
1. ZIPファイルを解凍後、任意の場所に配置してください。 ※ C:\Program Files など、ユーザーアカウント制御により書き込みが制限される場所は避けてください。
2. PunctuationSwitcher.exeをダブルクリックして実行してください。
3. ソフトウェアを終了する際は、タスクトレイアイコンを右クリックし、「終了」をクリックします。

アンインストール
フォルダーごと削除してください。
レジストリを書き換えることで句読点の設定を変更していますが、本ツール自体はレジストリを使用していません。
